# User Roles & Permissions

## Roles

1. **Admin**
   - Full access to all system functionalities.
   - Manage users and settings.

2. **Supervisor**
   - Manage tasks and oversee operations.
   - Access to reporting features.

3. **Health Worker**
   - Input and manage vaccination data.
   - Update patient records.

4. **Storekeeper**
   - Manage inventory and supplies.
   - Track stock levels.

5. **Viewer**
   - View access to records and reports.
   - No editing privileges.

## Permissions

### Admin
- Create, read, update, and delete all records.
- Manage system settings and configurations.

### Supervisor
- Read and update tasks and reports.
- Access to performance analytics.

### Health Worker
- Create and update patient records.
- View vaccination schedules.

### Storekeeper
- Update inventory logs.
- View supply orders.

### Viewer
- Read-only access to general data.
